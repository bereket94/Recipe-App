# Recipe App
A mobile recipe application built with React Native (or Expo, based on the app.json file). This app allows users to browse, search, and view detailed recipes. The project includes a clean architecture with separation of assets, app logic, and configuration files.

Features (suggested)
Browse recipes

View recipe details

Clean and responsive UI

Built with TypeScript for type safety

Project Structure
app/ - Main application code

assets/ - Images, fonts, and other static assets

.vscode/ - Editor-specific settings

Configuration files: app.json, tsconfig.json, eslint.config.js
